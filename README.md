- Tạo ra sản phẩm thi thử trắc nghiệm THPT Quốc gia
- Mục đích giúp các bạn học sinh kiểm tra được kiến thức đã học và đánh giá học lực của bản thân.

- Ngôn ngữ lập trình : PHP.
